# **App Name**: SBKTECH Clone

## Core Features:

- Homepage Product Display: Showcase smart security products categorized into 'Serrure intelligente', 'Pointeuse biométrique', and 'Coffre Fort' using responsive product cards featuring discounts and prices in Moroccan Dirham.
- Product Detail Pages with AI-Enhanced Descriptions: Provide a comprehensive view for each product, including large image galleries, star ratings, and detailed descriptions that have been enhanced by an AI tool for improved SEO and user engagement. Includes semantic HTML and JSON-LD schema placeholders.
- Streamlined Cash on Delivery Checkout: Implement a direct, inline purchase form on product pages for cash-on-delivery orders, collecting essential user details like first name, phone number, and city.
- Responsive Navigation & Layout: Integrate a sticky header with primary navigation (Accueil, Poignée Digital, Coffre-fort, Catalogue, Contact), search, user, and cart icons, complemented by a dark-themed footer with quick links and an email subscription form, ensuring full responsiveness for mobile-first access.
- SEO and Performance Foundation: Leverage Next.js Image component for optimized asset loading, enforce proper semantic HTML usage across all pages, and include a block for JSON-LD Product Schema markup on product detail pages for rich snippets in search results.

## Style Guidelines:

- Color Anchor: Modern, high-tech security, precision, cool tones. Inspired by 'Deep Black, Crisp White, and subtle Blue/Gray accents'. Dark color scheme. Background color: A very deep, almost black, subtly cool grey for the main interface areas (#1A1C1D). Primary color: A clear, modern tech blue to serve as a key accent and interactive element color (#4F85F5). Accent color: A complementary, slightly brighter aqua-cyan to provide highlight or secondary accent, analogous to the primary (#6EEAFB).
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, machined, and highly readable qualities, ideal for a high-tech security application. All content will be in French, enhancing readability and locale consistency.
- Utilize Lucide React icons throughout the application, specifically for the sticky header (search, user, cart), features bar (Empreinte, Clé, Code, Carte, APP), and other interactive elements, maintaining a clean, modern, and high-tech aesthetic.
- Implement a responsive, mobile-first grid system for all content sections, including category grids on the homepage. Strategic use of semantic HTML tags like <header>, <main>, <section>, <article>, and <footer> ensures structured content and robust SEO. Moroccan Dirham (MAD) is used as the currency.
- Integrate subtle hover effects and smooth transitions on interactive components such as navigation links, product cards, and buttons to provide engaging user feedback without detracting from performance.
